# SPEC -- Hermes Agent 增强版 Web 控制台

## 1. 目标与价值
- **解决什么问题**: 官方 Hermes 控制台 UI 简陋、英文界面、移动端体验差
- **为什么要做**: 中文友好、小白易用的 Web 控制台管理 Hermes Agent
- **预期收益**: 浏览器管理会话、定时任务、技能、模型、监控

## 2. 上下文与约束
- **技术栈**: Vue 3 (CDN) + ECharts + Tailwind CSS (CDN)，零构建依赖
- **现有架构**: Hermes Gateway 内部 API (端口 10086)，纯前端 SPA + serve.py 代理
- **约束**: 纯前端无构建，中文 UI，默认浅色主题

## 3. 功能需求

### F1: 仪表盘首页 (P0)
- 系统概览（CPU/内存/磁盘）、活跃会话数、定时任务状态、模型信息

### F2: 会话管理 (P0)
- 会话列表、详情、消息历史

### F3: 定时任务管理 (P0)
- 查看/创建/编辑/删除/暂停/恢复

### F4: 技能管理 (P1)
- 技能列表、启用/禁用

### F5: 模型配置 (P1)
- 当前模型、切换模型、Provider 配置

### F6: 消息渠道 (P1)
- 已连接平台状态

### F7: 系统日志 (P2)
- 日志查看、筛选搜索

### F8: 系统监控 (P2)
- 实时 CPU/内存/磁盘图表

### F9: 主题切换 (P1)
- 默认浅色，支持深色，localStorage 保存偏好

### F10: Gateway 启停控制 (P1)
- 启动/停止/重启 Gateway，官方控制台入口

## 4. 非功能需求

| 维度 | 要求 |
|------|------|
| 性能 | 首屏 < 2s |
| 安全 | Token 认证 |
| 兼容性 | 现代浏览器 + 移动端 |
