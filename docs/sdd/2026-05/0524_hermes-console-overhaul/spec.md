# SPEC — Hermes 控制台全面重构

## 1. 目标与价值

- **解决什么问题**：当前控制台 UI 极差、功能缺失，完全无法用于日常管理
- **为什么要做**：Hermes 需要一个专业的管理界面来提升运维效率
- **预期收益**：用户可以通过 Web 控制台完成所有 Hermes 管理操作

## 2. 上下文与约束

- **技术栈**：Vue 3 + ECharts + Tailwind CSS + Python HTTP Server
- **现有架构**：serve.py 后端代理 + 静态前端文件
- **依赖**：Gateway Internal API (localhost:18643)、本地文件系统
- **约束**：
  - 必须在 Docker 容器内运行
  - 不修改 Gateway 核心代码
  - 保持后端代理架构（无 CORS）
  - 默认浅色主题，支持切换深色

## 3. 功能需求

### F0: 基础框架重构
- **描述**：修复首次加载问题、移除弹窗、建立正确路由
- **优先级**：P0

### F1: 仪表盘 (Dashboard)
- **描述**：系统概览页面，展示关键指标和快速入口
- **功能**：
  - Gateway 状态卡片（运行中/已停止 + 重启按钮）
  - 系统资源监控（CPU/内存/磁盘进度条 + 百分比）
  - 已配置模型（当前模型名称 + 提供商）
  - 连接渠道（平台名称中文 + 连接状态 + 正确图标）
  - 最近日志（最近 10 条，带级别颜色标签）
  - 快速操作入口（跳转到各管理页面）
- **优先级**：P0

### F2: 会话管理 (Sessions)
- **描述**：管理所有对话会话
- **功能**：
  - 会话列表（分页、搜索、排序）
  - 会话详情（消息历史、时间线展示）
  - 会话编辑（重命名标题）
  - 会话删除（确认弹窗）
  - 会话来源标签（cli/feishu/telegram 等）
- **优先级**：P0

### F3: 定时任务 (Cron Jobs)
- **描述**：管理定时任务的完整 CRUD
- **功能**：
  - 任务列表（名称、计划、状态、最后执行时间）
  - 创建任务（表单：名称、计划表达式、prompt）
  - 编辑任务（修改计划、prompt、启用/禁用）
  - 删除任务（确认弹窗）
  - 立即执行（手动触发）
  - 执行历史（最近执行记录）
- **优先级**：P0

### F4: 技能管理 (Skills)
- **描述**：查看和管理已安装的技能
- **功能**：
  - 技能列表（卡片网格、搜索筛选）
  - 技能详情（名称、描述、版本、作者、文件路径）
  - 技能启用/禁用
  - 从 Skills Hub 搜索并安装（如果 API 可用）
- **优先级**：P1

### F5: 插件管理 (Plugins)
- **描述**：管理 MCP 服务器和工具集
- **功能**：
  - 插件列表（分类筛选、搜索）
  - 插件详情（类型、状态、配置）
  - 插件启用/禁用
  - 插件配置编辑
- **优先级**：P1

### F6: 模型配置 (Models)
- **描述**：管理和切换 AI 模型
- **功能**：
  - 当前模型展示（名称、提供商、API 地址）
  - 模型切换（下拉选择已有提供商 + 模型名称，非手动输入）
  - 提供商列表（名称、API 地址、密钥脱敏显示）
  - 添加/编辑/删除提供商
  - 模型验证（切换前测试连接）
- **优先级**：P0

### F7: 消息渠道 (Channels)
- **描述**：管理消息平台连接
- **功能**：
  - 渠道列表（平台中文名 + 正确图标 + 连接状态）
  - 渠道详情（配置信息、连接状态）
  - 渠道启用/禁用
  - 渠道配置编辑
- **优先级**：P1

### F8: 系统设置 (Settings)
- **描述**：全局配置管理
- **功能**：
  - 分类展示所有配置项
  - 每个配置项有中文说明
  - 配置修改 + 保存
  - 配置重置
  - 配置导入/导出
- **优先级**：P1

### F9: 系统监控 (Monitor)
- **描述**：系统和 Hermes 运行状态监控
- **功能**：
  - NAS 资源监控（CPU/内存/磁盘）
  - Hermes 监控（活跃会话数、Token 使用量、API 调用次数）
  - 趋势图表（ECharts 折线图）
  - 实时刷新（10 秒间隔）
- **优先级**：P1

### F10: 日志查看 (Logs)
- **描述**：查看和搜索系统日志
- **功能**：
  - 日志列表（时间、级别、消息）
  - 级别筛选（ERROR/WARN/INFO/DEBUG）
  - 关键词搜索
  - 日志级别颜色标签
  - 自动滚动
  - 日志导出
- **优先级**：P1

## 4. 非功能需求

| 维度 | 要求 | 说明 |
|------|------|------|
| 性能 | 首屏加载 < 3 秒 | 优化资源加载 |
| 响应式 | PC + 移动端适配 | 响应式布局 |
| 主题 | 浅色/深色双主题 | 默认浅色，可切换 |
| 浏览器 | Chrome/Firefox/Safari 最新版 | 现代 CSS 特性 |
| 可用性 | 所有操作有反馈 | Toast 通知、加载状态 |

| 编号 | 场景 | 预期结果 | 通过条件 |
|------|------|----------|----------|
| AC1 | 首次访问控制台 | 直接显示仪表盘，无弹窗 | PC 和移动端均无异常弹窗 |
| AC2 | 官方控制台链接 | 跳转到 http://192.168.5.110:19119/login | 点击后正确打开 |
| AC3 | 会话管理 - 查看列表 | 显示所有会话，ID 完整可见 | 分页正常，搜索可用 |
| AC4 | 会话管理 - 查看详情 | 显示消息历史，样式正常 | 浅色模式文字可见 |
| AC5 | 会话管理 - 重命名 | 可以修改会话标题 | 修改后立即生效 |
| AC6 | 定时任务 - 创建 | 可以创建新定时任务 | 任务出现在列表中 |
| AC7 | 定时任务 - 编辑 | 可以修改任务配置 | 修改后立即生效 |

| AC9 | 模型配置 - 切换 | 下拉选择提供商和模型 | 切换后对话使用新模型 |

## 6. 边界与异常

| 场景 | 处理方式 |
|------|----------|
| Gateway 未运行 | 显示"已停止"状态，提供启动按钮 |
| API 请求失败 | Toast 错误提示，不阻断页面 |
| 数据为空 | 显示空状态插画 + 引导文案 |
| 网络断开 | 显示离线提示 |
| 大量数据 | 分页加载，每页 20 条 |
